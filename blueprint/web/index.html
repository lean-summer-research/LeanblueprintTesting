0.1 Green’s preorders (from MyProject/GreensRelations.lean)

Definition 1 Green’s \(R\)-preorder
✓
#

Lean declarations

  • RRel

Let \(M\) be a monoid and \(x,y\in M\). We define \(x \le _R y\) iff there exists \(z\in M\) with \(x\cdot z = y\). This relation is reflexive and transitive.

  • RRel.op_iff

For \(x,y\in M\),

\[ RRel(\mathrm{op}\, x,\mathrm{op}\, y)\ \iff \ RRel(x,y). \]

Equivalently, the \(R\)-preorder on \(M\) corresponds to the \(R\)-preorder on \(\mathrm{MulOpposite}\, M\) via \(\mathrm{op}\).
